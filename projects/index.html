Central Processing Unit Simulator

Description

A program that simulates a CPU with x86 Intel core architecture. Structures in C will represent parts of an Intel CPU such as memory cache, decoder, control unit, registers, arithmetic/logic unit (ALU), and floating point unit (FPU). The program reads in an assembly language file created using the g++ -S option. The lines in the file are read into a struct Reader that assigns consecutive simulated memory locations to each line of the instructions, starting with address 100. The main() then starts a traditional fetch-decode-execute loop that will “execute” the assembly language file by displaying the contents of the registers after each assembly language instruction has been “executed”.

Details

  • Project for Computer Science course at the University of California, Davis
  • Different versions of the program written in both C and C++
  • Authors: Terry Yang and Derek Wong

Employee Database

Description

A GUI database that can store the information of employees in a company. The application features the ability to add employees into the database, search for existing employees using their employee numbers, editing the information on employees in the database and displaying a list of employees in the database.

Details

  • Project for Computer Science course at The Woodlands School
  • Written in Java using the Swing GUI Builder in the Netbeans IDE
  • Author: Terry Yang
